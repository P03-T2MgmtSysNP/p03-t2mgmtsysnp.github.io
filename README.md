# P03‑T2MgmtSysNP – Project Website

This repository hosts the public website for **Management System for Nonprofits** (CS 4850‑04 – Fall 2025).

## How to run (GitHub Pages)
1. Ensure the repository name is **p03-t2mgmtsysnp.github.io** and is **public** (inside the `P03-T2MgmtSysNP` organization).
2. Go to **Settings → Pages** and set:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** (root)
3. Visit: `https://p03-t2mgmtsysnp.github.io/`

## Edit Content
- Main page: `index.html`
- Styles: `css/styles.css`
- Add images to: `/img`
- Upload reports/slides into: `/docs` and link them in the **Documentation** section of `index.html`.
